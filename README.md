### Iniciar o projeto
- mkdir backend
- npm init -y
- npm install -g pnpm
- pnpm install express cors dotenv
- pnpm install nodemon -D
- pnpm install bcryptjs
- pnpm install jsonwebtoken

### em package.json e dentro dat tag scriprs, inserir
- "start": "nodemon app.js"

### Instalar Sequelize
